### Play against the engine or your friend locally in style.



* Challenge the computer to game at different levels
* Play a local opponent 'over-the-board', using the clock if you wish
* Load/export the board state in FEN
* Export your games to PGN
* Customise the colours and piece set



This was my A-level Computer Science programming project. The engine is my own simple minimax implementation, so it's not very powerful but you have to be careful on its highest setting (depth 6)!

Download the app to test it [here](https://drive.google.com/file/d/11GeF9KvovfNpo99vlcF0mcVf1s6T6CXi/view?usp=sharing) - extract all and run LeoChess.bat.
Note that the download is quite large (90MB compressed) due to the JVM being bundled.
